# 🌐🎤 Voice Recognition and Translation(VRT)

## 🧠 AI-Powered Multilingual Voice Recognition & Translation Web App

Voice recognition and Translation (VRT) is an advanced browser-based AI voice translation application that can:

- 🎤 Recognize speech in multiple languages
- 🌍 Detect and translate spoken text
- 🔈 Speak translated text using AI voice
- 🌓 Support dark/light themes
- 📋 Copy and download translated text
- 📝 Save translation history
- 🤖 Support basic voice commands
- 📱 Work on desktop and mobile browsers

This project is fully frontend-based and works directly in the browser without requiring any backend server.

---

# 🚀 Live Features

## 🎙️ Voice Recognition
- Real-time speech recognition
- Supports multiple Indian & international languages
- Auto language detection support

---

## 🌍 Translation System
- Instantly translates recognized speech
- Supports multilingual translation
- Uses Google Translate API

---

## 🔈 AI Voice Output
- Speaks translated text aloud
- Adjustable:
  - Voice speed
  - Voice pitch

---

## 🌓 Dark / Light Theme
- Toggle between dark and light modes
- Theme preference saved using Local Storage

---

## 📝 Translation History
- Stores recent translations inside the app
- Displays:
  - Original text
  - Translated text

---

## 📋 Copy & Download
- Copy translated text to clipboard
- Download translated text as `.txt` file

---

## 🔄 Language Swap
- Quickly swap input and output languages

---

## 🎧 Real-Time Mic Animation
- Animated listening waveform while recording voice

---

## 🤖 Voice Commands
Basic supported commands:
- “Dark mode”
- “Clear text”
